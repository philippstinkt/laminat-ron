# ready or not - laminat edition

### philipp stinkt


1. download the mods [here](https://github.com/philippstinkt/laminat-ron/releases/tag/laminat).

2. extract the mod

3. place the extracted .pak file in your paks folder. your paks folder will be located in ```C:\Program Files (x86)\Steam\steamapps\common\Ready Or Not\ReadyOrNot\Content\Paks``` or something like that depending on where you have steam and ready Or not installed.

4. launch the game and enjoy!
